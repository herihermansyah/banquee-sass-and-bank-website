## 🚀 Key Features
- **Cutting-Edge Stack**: Built with the latest Next.js 16.2 (App Router) and React 19.[cite: 1]
- **Next-Gen Styling**: Implemented using Tailwind CSS v4 for optimized performance and modern configurations.[cite: 1]
- **React Compiler**: Leveraged React 19’s latest features for superior rendering efficiency.[cite: 1]
- **Responsive & Pixel-Perfect**: Seamlessly adapted for mobile, tablet, and desktop views with a focus on high-fidelity design.[cite: 1]
- **Modular Components**: Structured using Radix UI and Shadcn for accessible and reusable UI elements.[cite: 1]

## 💻 Getting Started

To run this project locally, follow these steps:

1. **Clone the repository:**
   ```bash
   git clone [https://github.com/herihermansyah/banquee-sass-and-bank-website.git](https://github.com/herihermansyah/banquee-sass-and-bank-website.git)
   ```[cite: 1]

2. **Navigate to the directory:**
   ```bash
   cd banquee-sass-and-bank-website
   ```[cite: 1]

3. **Install dependencies:**
   ```bash
   npm install
   ```[cite: 1]

4. **Start the development server:**
   ```bash
   npm run dev
   ```[cite: 1]

5. **Open your browser:**
   Go to [http://localhost:3000](http://localhost:3000) to see the result.[cite: 1]
